# All Design Tokens

Complete reference of all design tokens in the Bioma design system.

## How to Use This Reference

Each token includes:
- **Token name**: CSS custom property name
- **Value**: Current value (light mode)
- **Usage**: When to use this token
- **Type**: Foundation or Semantic

**Color preview squares** show the actual color for visual reference.

---
