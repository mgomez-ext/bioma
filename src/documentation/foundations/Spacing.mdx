# Espaciado

El espaciado consistente crea ritmo visual y jerarquía en las interfaces de Bioma.

## Escala de Espaciado

Basada en un **sistema de grilla de 4px**:

| Token | Valor | Uso |
|-------|-------|-------|
| --spacing-0 | 0px | Sin espacio |
| --spacing-1 | 4px | Espacios mínimos |
| --spacing-2 | 8px | Espacios pequeños |
| --spacing-3 | 12px | Espacios por defecto |
| --spacing-4 | 16px | Espacios medianos |
| --spacing-5 | 20px | Espacios grandes |
| --spacing-6 | 24px | Espacios entre secciones |
| --spacing-8 | 32px | Secciones grandes |
| --spacing-10 | 40px | Secciones XL |
| --spacing-12 | 48px | Secciones XXL |
| --spacing-16 | 64px | Secciones de página |

## Uso

### Espaciado de Componentes

```scss
.card {
  padding: var(--spacing-6);    // Espaciado interno
  margin-bottom: var(--spacing-8);  // Espaciado externo
  gap: var(--spacing-4);         // Entre elementos
}
```

### Espaciado de Layout

```scss
.container {
  padding: var(--spacing-4);

  @media (min-width: 768px) {
    padding: var(--spacing-6);
  }

  @media (min-width: 1024px) {
    padding: var(--spacing-8);
  }
}
```

## Mejores Prácticas

- Usá tokens de espaciado de forma consistente
- Preferí espacios más grandes entre secciones
- Usá espacios más pequeños dentro de componentes
- Mantené el ritmo vertical
