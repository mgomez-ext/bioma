# Spacing

Consistent spacing creates visual rhythm and hierarchy in Bioma interfaces.

## Spacing Scale

Based on a **4px grid system**:

| Token | Value | Usage |
|-------|-------|-------|
| --spacing-0 | 0px | No space |
| --spacing-1 | 4px | Tiny gaps |
| --spacing-2 | 8px | Small gaps |
| --spacing-3 | 12px | Default gaps |
| --spacing-4 | 16px | Medium gaps |
| --spacing-5 | 20px | Large gaps |
| --spacing-6 | 24px | Section gaps |
| --spacing-8 | 32px | Large sections |
| --spacing-10 | 40px | XL sections |
| --spacing-12 | 48px | XXL sections |
| --spacing-16 | 64px | Page sections |

## Usage

### Component Spacing

```scss
.card {
  padding: var(--spacing-6);    // Internal spacing
  margin-bottom: var(--spacing-8);  // External spacing
  gap: var(--spacing-4);         // Between elements
}
```

### Layout Spacing

```scss
.container {
  padding: var(--spacing-4);

  @media (min-width: 768px) {
    padding: var(--spacing-6);
  }

  @media (min-width: 1024px) {
    padding: var(--spacing-8);
  }
}
```

## Best Practices

- Use spacing tokens consistently
- Prefer larger gaps between sections
- Use smaller gaps within components
- Maintain vertical rhythm
