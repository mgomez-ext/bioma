# Color

El color es uno de los elementos de diseño más impactantes en Bioma. Nuestro sistema de color proporciona una paleta completa diseñada para claridad, accesibilidad y resonancia emocional.

## Filosofía del Color

El sistema de color de Bioma se construye sobre tres principios:

1. **Orientado a Propósito**: Cada color tiene un propósito y significado claros
2. **Accesible**: Todas las combinaciones de color cumplen con los estándares WCAG AA
3. **Adaptativo**: Los colores funcionan hermosamente tanto en modos claro como oscuro

## Paleta de Marca

**Rojo Bioma** (#E30038) es nuestro color de marca primario, representando energía, pasión y acción.

Mirá la [referencia completa de tokens](#) para todos los valores de color y guías de uso.

## Colores Semánticos

### Colores de Texto
- `--text-strong` - Encabezados y énfasis
- `--text-primary` - Texto de cuerpo
- `--text-secondary` - Texto de soporte
- `--text-link` - Enlaces interactivos

### Colores de Fondo
- `--bg-default` - Fondos primarios
- `--bg-subtle` - Fondos secundarios
- `--bg-inverse` - Fondos oscuros

### Colores de Acción
- `--primary-default` - Acciones primarias (rojo de marca)
- `--primary-hover` - Estados hover
- `--primary-active` - Estados activos
