# Tipografía

La tipografía es esencial para crear interfaces claras, legibles y visualmente atractivas en Bioma.

## Familia Tipográfica

### Fuente Primaria: Sora

Sora es una sans-serif geométrica diseñada para interfaces digitales.

```scss
--font-primary: 'Sora', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

**Pesos Disponibles**: 300 (Light), 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)

### Fuente Monoespaciada: SF Mono

Para código y contenido técnico:

```scss
--font-mono: 'SF Mono', Monaco, 'Cascadia Code', monospace;
```

## Escala Tipográfica

| Token | Tamaño | Uso |
|-------|------|-------|
| --font-xs | 12px | Letra pequeña, subtítulos |
| --font-sm | 14px | Texto secundario, etiquetas |
| --font-base | 16px | Texto de cuerpo (predeterminado) |
| --font-lg | 18px | Texto de cuerpo grande |
| --font-xl | 20px | Subtítulos |
| --font-2xl | 24px | Títulos de sección |
| --font-3xl | 30px | Títulos de página |
| --font-4xl | 36px | Encabezados hero |
| --font-5xl | 48px | Encabezados display |

## Pesos de Fuente

```scss
--font-light: 300;
--font-regular: 400;
--font-medium: 500;
--font-semibold: 600;
--font-bold: 700;
```

## Alturas de Línea

```scss
--leading-none: 1;
--leading-tight: 1.25;
--leading-snug: 1.375;
--leading-normal: 1.5;
--leading-relaxed: 1.625;
--leading-loose: 2;
```

## Uso

### Encabezados

```scss
h1 {
  font-size: var(--font-4xl);
  font-weight: var(--font-bold);
  line-height: var(--leading-tight);
  color: var(--text-strong);
}

h2 {
  font-size: var(--font-3xl);
  font-weight: var(--font-semibold);
}
```

### Texto de Cuerpo

```scss
p {
  font-size: var(--font-base);
  font-weight: var(--font-regular);
  line-height: var(--leading-relaxed);
  color: var(--text-primary);
}
```
