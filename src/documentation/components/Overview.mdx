## Principios de Componentes

Todos los componentes de Bioma siguen estos principios:

- **Accesibles**: Cumplen con WCAG AA con etiquetas ARIA apropiadas
- **Basados en Tokens**: Usan design tokens exclusivamente (sin valores hardcodeados)
- **Standalone**: Componentes standalone de Angular para mejor tree-shaking
- **Tipados**: Soporte completo de TypeScript con tipos estrictos
- **Documentados**: Documentación completa en Storybook
