# Inicio Rápido

Creá tu primer componente con Bioma en 5 minutos.

## Paso 1: Creá un Nuevo Componente

```typescript
// app/my-card.component.ts
import { Component } from '@angular/core';
import { ButtonComponent, IconFontComponent } from '@mgomezw/bioma';

@Component({
  selector: 'app-my-card',
  standalone: true,
  imports: [ButtonComponent, IconFontComponent],
  templateUrl: './my-card.component.html',
  styleUrls: ['./my-card.component.scss']
})
export class MyCardComponent {
  handleClick() {
    console.log('Button clicked!');
  }
}
```

## Paso 2: Agregá el Template

```html
<!-- my-card.component.html -->
<div class="card">
  <div class="card__header">
    <bio-icon-font iconName="notifications" size="24" type="rounded"></bio-icon-font>
    <h2 class="card__title">Welcome to Bioma</h2>
  </div>

  <p class="card__content">
    Empezá a construir experiencias hermosas con el design system de Bioma.
  </p>

  <div class="card__actions">
    <bio-button
      label="Comenzar"
      variant="filled"
      size="large"
      [iconEnd]="true"
      iconEndName="arrow_forward"
      (clicked)="handleClick()">
    </bio-button>
    <bio-button
      label="Saber Más"
      variant="outlined"
      size="large">
    </bio-button>
  </div>
</div>
```

## Paso 3: Estilizá con Tokens

```scss
// my-card.component.scss
.card {
  background-color: var(--bg-default);
  border: 1px solid var(--border-default);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
  box-shadow: var(--shadow-md);
  max-width: 400px;

  &__header {
    display: flex;
    align-items: center;
    gap: var(--spacing-3);
    margin-bottom: var(--spacing-4);

    bio-icon-font {
      color: var(--icon-primary);
    }
  }

  &__title {
    font-family: var(--font-primary);
    font-size: var(--font-2xl);
    font-weight: var(--font-semibold);
    color: var(--text-strong);
    margin: 0;
  }

  &__content {
    color: var(--text-secondary);
    line-height: var(--leading-relaxed);
    margin-bottom: var(--spacing-6);
  }

  &__actions {
    display: flex;
    gap: var(--spacing-3);
  }
}
```

## Paso 4: Usá tu Componente

```typescript
// app.component.ts
import { Component } from '@angular/core';
import { MyCardComponent } from './my-card.component';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [MyCardComponent],
  template: `<app-my-card></app-my-card>`
})
export class AppComponent {}
```

## Qué Aprendiste

### Componentes
- Cómo importar y usar componentes de Bioma
- Pasar props a componentes
- Manejar eventos

### Design Tokens
- Usar propiedades personalizadas de CSS
- Tokens semánticos vs foundation tokens
- Tokens de color, espaciado y tipografía

### Mejores Prácticas
- Componentes standalone
- Estilos basados en tokens (sin valores hardcodeados)
- Espaciado y sizing consistentes

