# Instalación

Comienza a usar Bioma 2.0 en tu proyecto de Angular.

## Requisitos Previos

- Node.js 18+
- Angular 21+
- npm o yarn

## Instalar vía npm

```bash
npm install @mgomezw/bioma
```

O con yarn:

```bash
yarn add @mgomezw/bioma
```

## Configuración

### 1. Importar Estilos

Agrega los estilos de Bioma a tu `angular.json`:

```json
{
  "styles": [
    "node_modules/@mgomezw/bioma/styles/tokens.css",
    "src/styles.scss"
  ]
}
```

### 2. Importar Fuentes

Agrega a tu `index.html`:

```html
<head>
  <link href="https://fonts.googleapis.com/css2?family=Sora:wght@300;400;500;600;700&display=swap" rel="stylesheet">
  <link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Rounded:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200&display=swap" rel="stylesheet">
</head>
```

### 3. Importar Componentes

Los componentes son standalone y pueden importarse individualmente:

```typescript
import { Component } from '@angular/core';
import { ButtonComponent } from '@mgomezw/bioma';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [ButtonComponent],
  template: `
    <bio-button label="Hacé clic" variant="filled"></bio-button>
  `
})
export class AppComponent {}
```

## Usando Design Tokens

Los tokens están disponibles como propiedades CSS personalizadas:

```scss
.mi-componente {
  background-color: var(--primary-default);
  color: var(--text-primary);
  padding: var(--spacing-4);
  border-radius: var(--radius-md);
}
```

## Modo Oscuro

Habilitá el modo oscuro agregando el atributo `data-theme`:

```html
<body data-theme="dark">
  <!-- Tu aplicación -->
</body>
```

Alternar programáticamente:

```typescript
document.body.setAttribute('data-theme', 'dark');
// o
document.body.setAttribute('data-theme', 'light');
```

## Soporte para TypeScript

Bioma incluye definiciones completas de TypeScript:

```typescript
import { ButtonSize, ButtonVariant } from '@mgomezw/bioma';

const tamaño: ButtonSize = 'large';
const variante: ButtonVariant = 'filled';
```

## Solución de Problemas

### Los estilos no se cargan
Verificá que el archivo tokens.css esté importado en angular.json y que la ruta sea correcta.

### Las fuentes no se muestran
Asegurate de que los links de Google Fonts estén en tu index.html y que la solicitud de red sea exitosa.

### No se encuentran los componentes
Asegurate de que @mgomezw/bioma esté instalado e importado correctamente en tu componente.
